Object Oriented Programming

객체지향 프로그래밍의 근본적인 개념들인 클래스, 멤버변수, 멤버함수, 데이터 추상화, 인캡슐레이션, 상속성, 동적바인딩 등을 상세히 학습한다.

This course introduces fundamental concepts about object-oriented programming such as class, member variable, member function, data abstraction, encapsulation, inheritance and dynamic binding.
