기능목록

저희 No Name조에서 기획한 프로젝트인 '익명성을 강조한 설문조사 시스템'의 주요 기능은 크게 7가지입니다.

1. 회원가입, 비밀번호 암호화 기능(익명성)

1.1 회원가입 기능을 제공합니다.

1.2 회원가입 시 비밀번호를 암호화하여 익명성을 제공합니다.

2. 로그인

2.1 로그인 기능을 제공합니다.

3. 설문조사 기능

3.1 설문 조사를 제공합니다.

4. 설문조사 생성 기능(마감일, 참여 인원 설정 등) / (프로젝트 기획 시 변경 및 생략 가능)

4.1 현재 생성된 설문조사가 없을 시 설문조사를 생성할 수 있습니다.

5. 설문조사 결과 데이터 암호화 기능(익명성)

5.1 설문조사 결과 데이터를 암호화하여 데이터베이스에 저장되어 익명성을 보장합니다.

6. 설문조사 참여율 표시 기능

6.1 설문조사 참여율을 표시합니다.

7. 설문조사 결과 확인 기능

7.1 설문조사 결과를 확인할 수 있습니다.
