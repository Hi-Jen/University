■ 작업물 확인

최종 결과보고서를 작성하며 훗날 자기소개서 작성이나 발표 자료 등에 활용할 수 있도록 작성하였습니다.

아래는 해당 프로젝트의 파일 구조와 문서 내용을 요약한 페이지입니다.

■ 작업량 테이블

작성된 소스코드는 파일명별로 분류되어 있으며, HTML, CSS, PHP의 종류로 나뉩니다.

1. 소스코드

구성 파일:

  • main.html - 프로그램 시작 페이지
  • style2.css - main.html의 스타일 시트 페이지
  • add_post.php - 설문을 만들기 위한 페이지
  • delete_post.php - 설문을 삭제하기 위한 페이지
  • home.php - 로그인 후의 기본 화면이자 설문 목록을 보여주는 페이지
  • login.php - 로그인 기능을 구현 하기 위한 페이지
  • logout.php - 로그아웃 기능을 구현 하기 위한 페이지
  • style.css - 전반적인 소스 코드들의 스타일 시트 페이지
  • register.php - 회원가입 기능을 구현하기 위한 페이지
  • view_post.php - add_post에서 만든 설문을 확인하고 참여할 수 있게 만든 페이지
  • view_result.php - view_post에서 행한 설문의 결과를 정리한 페이지
