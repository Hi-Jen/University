- 데이터 설명 -

■ 익명성을 강조한 설문조사 시스템

  1. 회원가입을 합니다.
    • 학번(아이디), 비밀번호, 이름을 받습니다.
    • 사람입니까? 문구의 클릭 버튼을 눌러야만 회원가입 가능. (기능 생략 가능)
  2. 회원가입한 데이터는 데이터베이스에 저장됩니다.
    • 이미 회원가입되어있는 데이터가 있을 시에는 오류메세지를 출력합니다.
  3. 저장된 데이터를 기반으로 로그인합니다.
    • 로그인 실패 5회시 5분간 로그인을 금지. (기능 생략 가능)
  4. 로그인 후, 참여 가능한 설문이 있으면 참여합니다.
    • 설문조사가 많이 있을시에 주제, 기간별로 설문을 찾아볼 수 있습니다. (기능 생략 가능)
  5. 참여 가능한 설문이 없으면 새로운 설문을 만듭니다.
    • 설문조사 질문 설정
    • 설문조사 기간 지정
    • 설문조사 참여인원 설정
  6. (만약 데이터 베이스에 이미 저장된 설문조사 방식일 경우)
    • 설문조사 생성 페이지 생략
    • 설문조사 참여자의 설문 데이터만 암호화
  7. 설문조사 마감일이 지나면 설문조사 종료 페이지로 이동합니다.
    • 설문조사에 일정 인원이 답변하였을때 종료되거나, 설문작성자가 임의로 설문조사를 종료할 수 있습니다. (기능 생략 가능)
  8. 설문조사 결과는 데이터베이스에 익명으로 암호화되어 저장됩니다.
    • 설문조사 참여 시 설문자의 설문 참여 사실은 데이터 베이스에 저장되어 설문자가 설문에 참여했다는 사실은 알 수 있습니다.
    • 설문조사 데이터는 익명으로 암호화되어 저장되며 익명 설문 데이터의 결과만 도출하여 설문조사 결과를 결과 페이지에 표시합니다.
  9. 설문조사 마감 후, 결과 페이지가 표시됩니다.
    • 설문조사 암호화 데이터 결과를 합산하여 결과 페이지에 도출합니다.
  10. 결과 페이지에서 버튼을 클릭하면 설문 시스템 사이트로 돌아갈 수 있습니다.
    • 설문조사 결과는 퍼센테이지 방식이나 표나 그래프 형식으로 표시됩니다.
