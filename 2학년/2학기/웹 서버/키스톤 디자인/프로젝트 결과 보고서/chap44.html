■ 기능 목록

■ 익명 설문 시스템 주요 기능

  1. 회원가입 및 비밀번호 암호화
    2.
    • 1.1 회원가입 기능 제공 (아이디, 비밀번호, 이름 입력)
    • 1.2 동일한 아이디로 회원가입 시 오류 메시지 출력
    • 1.3 비밀번호 암호화 저장으로 사용자 정보 보호 및 익명성 보장

  2. 로그인
    3.
    • 2.1 로그인 기능 제공
    • 2.2 입력된 아이디와 비밀번호로 사용자 신원 확인

  3. 설문조사 목록
    4.
    • 3.1 설문 목록 확인 및 참여 가능
    • 3.2 로그인한 사용자의 정보 확인 가능
    • 3.3 로그아웃 기능 제공
    • 3.4 로그인 사용자가 작성한 설문 삭제 가능

  4. 설문조사 참여
    5.
    • 4.1 사용자가 설문에 응답 가능 (찬성, 중립, 반대 선택)
    • 4.2 중복 응답 방지 기능 제공
    • 4.3 설문 데이터는 암호화되어 익명성을 유지

  5. 설문조사 생성
    6.
    • 5.1 새로운 설문 작성 가능 (설문 제목, 질문 입력)
    • 5.2 기본 응답 옵션으로 찬성, 중립, 반대를 제공
    • 5.3 설문 생성 후 설문 목록에 추가

  6. 설문조사 데이터 암호화
    7.
    • 6.1 설문 응답 데이터를 암호화하여 데이터베이스에 저장
    • 6.2 암호화된 설문 데이터는 익명으로 처리
    • 6.3 암호화된 데이터를 기반으로 결과를 합산하여 표시

  7. 설문조사 결과 확인
    8.
    • 7.1 설문 종료 후 결과 페이지에서 결과 확인 가능
    • 7.2 설문 결과는 퍼센트, 표, 그래프 등으로 시각화하여 제공
    • 7.3 결과 페이지에서 홈으로 이동 가능한 버튼 제공
