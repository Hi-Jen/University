■ 설문조사 결과 페이지

■ 설문조사 결과를 암호화된 데이터를 합산하여 표시하는 페이지입니다.

  • 설문 데이터 암호화
    •
    • 사용자의 설문 참여 사실은 데이터베이스에 저장되고 이를 통해 설문에 참여했는지 여부를 확인할 수 있습니다.
    • 설문 데이터는 암호화되어 저장되며, 익명 데이터의 결과만 도출하여 결과 페이지에 표시됩니다.
  • 설문 종료 후 설문 페이지를 눌러 설문 결과 페이지로 이동합니다.
    •
    • 설문 목록페이지에서 참여한 설문의 하이퍼링크를 누릅니다.
    • 이미 참여한 설문은 설문 대신 '이미 투표하셨습니다.' 라는 문구를 확인할 수 있습니다.
    • 설문 페이지로 들어가 결과보기 버튼을 통해 설문 결과를 확인할 수 있습니다.
  • 설문 마감 후 결과 페이지
    •
    • 암호화된 설문 데이터의 결과를 합산하여 결과 페이지에 표시합니다.
    • 설문 결과는 퍼센테이지 방식, 그래프 형식으로 제공됩니다.
    • 결과 페이지에서 스크롤을 내려 게시글로 돌아가기, 목록으로 돌아가기 버튼을 클릭하면 해당 페이지로 돌아갑니다.
