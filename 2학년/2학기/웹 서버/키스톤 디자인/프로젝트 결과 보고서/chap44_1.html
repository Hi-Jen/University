■ 회원가입 및 비밀번호 암호화 기능

  • 회원가입 기능을 제공합니다.

    • 사용자는 아이디(학번), 비밀번호, 이름을 입력해야 합니다.
    • 입력된 데이터는 데이터베이스에 저장되며, 비밀번호는 암호화되어 저장됩니다.
  • 동일한 아이디(학번)로 회원가입 시, 오류 메시지가 출력되는 기능을 제공합니다.

    • 사용자는 자신의 아이디(학번)가 이미 등록되었는지 확인할 수 있습니다.

  • 비밀번호 암호화 기능을 제공합니다.

    • 비밀번호는 암호화되어 데이터베이스에 안전하게 저장됩니다.
    • 암호화를 통해 익명성 및 데이터 신뢰성을 강화합니다.
  • - 비밀번호 암호화 방법 -
    •
    • 1. MySQL 내장 함수 사용:
      - PASSWORD()나 SHA2 함수를 사용합니다.

    • 2. PHP에서 bcrypt 사용:
      - password_hash()와 password_verify()를 통해 강력한 암호화와 검증이 가능합니다.
  • PHP 코드 예시:
    •
    • 1. 비밀번호 암호화 저장 코드: 
      
            $hashed_pass = password_hash($pass, PASSWORD_DEFAULT);
            mysqli_stmt_bind_param($stmt, "sss", $id, $hashed_pass, $name);    
            // password_hash(): PHP에서 제공하는 함수로, 비밀번호를 해싱(암호화)하여 안전하게 저장할 수 있도록 합니다.
    • 2. 비밀번호 검증 코드: 
      
            if (password_verify($input_pass, $hashed_pass)) {
                // 비밀번호가 일치하는 경우의 처리
            } else {
                // 비밀번호 불일치 처리
            }            
            // password_hash()와 password_verify()를 통해 강력한 암호화와 검증이 가능합니다.
