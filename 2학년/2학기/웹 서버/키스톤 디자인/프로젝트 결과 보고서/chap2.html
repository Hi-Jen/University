■ 데이터 설명

■ 익명 설문 시스템

  1. 시스템 시작 페이지에서 회원가입 버튼을 눌러 회원가입을 합니다.
    • 아이디(학번), 비밀번호, 이름을 입력받습니다.
    • 회원가입한 데이터는 데이터베이스에 저장됩니다.
    • 비밀번호는 암호화되어 저장됩니다.

  2. 이미 동일한 아이디(학번)로 회원가입된 경우, 오류 메시지를 출력합니다.
    • 자신의 아이디(학번)가 이미 회원가입되어 있는지 확인할 수 있습니다.

  3. 저장된 데이터를 기반으로 로그인합니다.
    • 아이디(학번)와 비밀번호를 입력하고 '로그인' 버튼을 클릭하여 인증 절차를 진행합니다.
    • 비밀번호가 잘못된 경우, 오류 메시지를 출력합니다.

  4. 로그인 후, 목록 페이지에서 참여 가능한 설문이 있는지 확인합니다.
    • 목록 페이지의 좌측 상단에서 로그인한 사용자를 확인할 수 있습니다.
    • 목록 페이지의 우측 상단에서 로그아웃 버튼을 눌러 로그아웃할 수 있습니다.
    • 사용자는 제목란의 하이퍼링크 텍스트를 눌러 설문에 참여할 수 있습니다.
    • 로그인한 사용자가 작성한 설문은 삭제 버튼을 통해 삭제할 수 있습니다.
    • 작성한 설문 삭제는 해당 설문 작성자만 가능합니다.

  5. 사용자가 설문에 참여할 수 있습니다.
    • 사용자는 설문에 응답 후 찬성, 중립, 반대의 의견 중 하나를 선택하고 제출 버튼을 클릭합니다.
    • 설문 제출 시 설문자의 결과 데이터는 암호화되어 데이터베이스에 저장됩니다.

  6. 참여한 설문 결과는 암호화되어 저장됩니다.
    • 설문 참여 시 설문자의 참여 사실은 데이터베이스에 저장되며, 설문자가 설문에 참여했는지 알 수 있습니다.
    • 설문 데이터는 암호화되어 저장되며, 결과만 도출되어 결과 페이지에 표시됩니다.

  7. 참여 가능한 설문이 없으면 새로운 설문을 만듭니다.
    • 목록 페이지에서 설문 작성 버튼을 눌러 설문 작성 페이지로 이동합니다.
    • 설문을 작성할 때 사용자는 설문 제목과 질문(내용)을 입력합니다.
    • 설문 선택지는 기본적으로 찬성, 중립, 반대의 3가지로 구성됩니다.
    • 설문 작성 후 게시글 추가 버튼을 통해 설문을 추가할 수 있습니다.

  8. 설문 참여 후 참여한 설문 페이지를 눌러 설문 결과 페이지로 이동합니다.
    • 설문 목록페이지에서 참여한 설문의 하이퍼링크를 누릅니다.
    • 참여한 설문은 설문 대신 '이미 투표하셨습니다.' 라는 문구를 확인할 수 있습니다.
    • 설문 페이지로 들어가 결과보기 버튼을 통해 설문 결과를 확인할 수 있습니다.

  9. 설문 마감 후, 결과 페이지가 표시됩니다.
    • 암호화된 설문 데이터의 결과를 합산하여 결과 페이지에 도출합니다.
    • 설문 결과는 퍼센트 방식 또는 그래프 형식으로 표시됩니다.
    • 결과 페이지에서 게시글로 돌아가기 및 목록으로 돌아가기 버튼을 클릭하면 해당 페이지로 돌아갑니다.
