■ 화면 설계

■ 익명 설문 시스템

익명 설문 시스템 시작 페이지

  1. 회원가입을 합니다.


    • 아이디(학번), 비밀번호, 이름을 입력받습니다.
    • 작성 후, 회원가입 버튼을 눌러 회원가입합니다.

  2. 회원가입한 데이터는 암호화되어 데이터베이스에 저장됩니다.


    • 비밀번호를 암호화하여 익명성을 보장합니다.

  3. 이미 동일한 아이디(학번)로 회원가입된 경우, 오류 메시지를 출력합니다.


    • 자신의 아이디(학번)이 이미 회원가입되어있는지 확인할 수 있습니다.

  4. 저장된 데이터를 기반으로 로그인합니다.


    • 아이디(학번)와 비밀번호를 입력하고 '로그인' 버튼을 클릭하여 인증 절차를 진행합니다.

  5. 로그인 후, 목록페이지에서 참여 가능한 설문이 있는지 확인합니다.


    • 목록페이지의 좌측상단에서 로그인한 사용자를 확인할 수 있습니다.
    • 목록페이지의 우측상단에서 로그아웃 버튼을 눌러 로그아웃할 수 있습니다.
    • 사용자는 제목칸의 하이퍼링크 텍스트를 눌러 설문에 참여할 수 있습니다.
    • 이때, 로그인한 사용자가 작성한 설문은 삭제 버튼을 통해 삭제할 수 있습니다.
    • 작성한 설문 삭제는 해당 설문 작성자만 가능합니다.

  6. 사용자가 설문에 참여할 수 있습니다.


    • 사용자는 설문에 응시한 후 찬성, 중립, 반대의 의견 중 하나를 선택한 후 제출버튼을 누릅니다.
    • 설문조사 제출 시 설문자의 설문조사 결과 데이터는 암호화되어 데이터베이스에 저장됩니다.

  7. 설문 결과는 익명으로 암호화되어 저장됩니다.

    • 설문 참여 시 설문자의 설문 참여 사실은 데이터 베이스에 저장되어 설문자가 설문에 참여했다는 사실은 알 수 있습니다.
    • 설문 데이터는 익명으로 암호화되어 저장되며 익명 설문 데이터의 결과만 도출하여 설문 결과를 결과 페이지에 표시합니다.

  8. 참여 가능한 설문이 없으면 새로운 설문을 만듭니다.


    • 목록페이지에서 설문조사 작성버튼을 통해 설문 작성페이지로 이동합니다.
    • 설문을 작성할 때, 사용자는 설문 제목, 질문(내용) 등을 입력합니다.
    • 설문 선택지는 기본적으로 찬성, 중립, 반대의 3가지로 구성됩니다.
    • 설문을 작성한 후 게시글 추가버튼을 통해 설문을 추가할 수 있습니다.

  9. 설문이 마감되면 목록페이지에서 설문조사 결과페이지로 이동합니다.

    • 설문 목록페이지에서 설문 결과확인란의 결과보기 버튼을 통해 설문 결과를 확인할 수 있습니다.

  10. 설문 마감 후, 결과 페이지가 표시됩니다.

    • 예시 이미지


    • 암호화된 설문 데이터의 결과를 합산하여 결과 페이지에 도출합니다.
    • 설문 결과는 퍼센테이지 방식이나 표나 그래프 형식으로 표시됩니다.
    • 결과 페이지에서 홈 버튼을 클릭하면 설문 목록 페이지로 돌아갑니다.
